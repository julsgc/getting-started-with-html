THE WEB
The web is the network of collected documents saved on servers and an individuals computers - all connected to one another, consisting of:

URLS
The Uniform Resource Locator is the file location or address of a file on the web (it is the www.blahblahblah.ninja for example).
HTTP
The Hypertext Transfer Protocol is a standard that allows users to exchange information found on webpages. It directs your computer over the internet to the servers where the document needed for your web page is stored. There is also FTP and HTTPS.
WEB APPLICATIONS
Any software/application that runs in a web browser, created in various languages (Java, CSS and HTML, for example)

A WEBPAGE
is a text document written in a web application language interpreted by your web browser to display them

HTML VOCABULARY
HTML is a web application language used to make web pages and consists of:
TAGS -
An HTML tag is always contained within angled brackets.
CLOSING - ones that require both a < > and < / > for example
VOID - these do not require a closing slash tag
ELEMENTS -
Everything within a set of opening and closing tags.
INLINE - keep information in the same line
BLOCK -separating paragraphs, creating blocks of information
ATTRIBUTE
The property of an HTML element. They sit inside the tag within quotations marks.
For example the link to www.studiograce.net looks like studio grace rocks

CODING WORKS LIKE THIS
Coding happens when programmers write text in a language that a computer can understand. The computer can then follow the instructions the programmer wrote. For example the computer might do this by making text like this: I'm learning to code!

COMPUTERS ARE STUPID
Programmers need to write exactly the way a computer understands (also known as writing with correct "syntax").
For example, if you forget to close a tag, the computer won't be able to figure out what you had intended to make bold.
This "stupidity" can be very frustrating, but it also gives programmers incredible power: if you know how to talk to a computer than you can tell it to do anything you want.
